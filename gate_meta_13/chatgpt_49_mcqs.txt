First, we need to determine the diffusion coefficient $D$ for carbon in austenite at the carburizing temperature. We can use the equation:

$D = D_0 \exp\left(\frac{-Q}{RT}\right)$

where $R$ is the gas constant and $T$ is the absolute temperature. Assuming a carburizing temperature of 927Â°C (1200 K), we get:

$D = 2.0 * 10^{-5} \exp\left(\frac{-142 \times 10^3}{8.314 \times 1200}\right) = 3.3 * 10^{-14} m^2/s$

Next, we can use Fick's second law to relate the concentration profile to the diffusion coefficient:

$\frac{\partial c}{\partial t} = D \frac{\partial^2 c}{\partial x^2}$

We can assume a one-dimensional case where the concentration varies only with depth, and use the boundary conditions:

$c(0,t) = 1.2$ wt.% (at the surface)

$c(x,t) = 0.2$ wt.% (at the initial depth)

$c(x,t_2) = 0.4$ wt.% (at the final depth after doubling)

We are looking for the time $t_2$ it takes to double the depth at which $0.4$ wt.% C is reached. This means the final depth $x_2$ is twice the initial depth $x_1$. We can write:

$\frac{x_2}{x_1} = 2$

$x_2 = 2x_1$

We can use the error function solution to Fick's second law for the concentration profile:

$c(x,t) = C_1 + (C_2 - C_1)\text{erf}\left(\frac{x}{2\sqrt{Dt}}\right)$

where erf is the error function and $C_1$ and $C_2$ are constants determined by the boundary conditions. With some algebraic manipulations, we can find that:

$\frac{C_1 - 0.2}{C_1 - 1.2} = e^{-\frac{x_1^2}{4Dt}}$

$\frac{C_2 - 0.2}{C_2 - 1.2} = e^{-\frac{x_2^2}{4Dt}}$

Substituting $x_2 = 2x_1$ and solving for $C_2/C_1$, we get:

$\frac{C_2}{C_1} = \frac{1.2 + 0.2e^{\frac{-x_1^2}{16Dt}}}{0.2 + 1.2e^{\frac{-x_1^2}{16Dt}}}$

Using this equation and the given ratio $x_2/x_1 = 2$, we can solve for $t_2/t_1$:

$\frac{t_2}{t_1} = \left(\frac{2\sqrt{D}x_1}{\sqrt{\pi}}\right)^2 \ln\left[\frac{1.2+0.2e^{-\frac{x_1^2}{16Dt_2}}}{0.2+1.2e^{-\frac{x_1^2}{16Dt_2}}}\right]$

We can use trial and error or numerical methods to solve this equation for $t_2/t_1$. Using a calculator, we get:

$\frac{t_2}{t_1} \approx 2.73$

Therefore, the time it takes to double the depth at which 0.4 wt.% C is reached is about 1.73 times the initial time. If we assume the initial time is 10 hours, the answer would be:

$t_2 = 2.73 \times 10 \approx 27.3 \approx 28$ hours
