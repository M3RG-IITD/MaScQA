

Using the formula,
Stress  = (Force) / (Area)
We can solve for stress.
Stress = 20 kN / 8 $cm^2$
Stress = 2.5 kN/$cm^2$

Deformation in steel,
$\varepsilon_{Stress}  = $ (Stress) / ($E_{steel}$)
$\varepsilon_{Stress}  = 2.5 / 200 x 10^3$
$\varepsilon_{Stress}  = 12.5 x 10^{-6}$

Let the final length of steel bar after deformation be L.

Using the formula,
$\Delta$(Length) = ($\Delta$(Length)$_x$ + $\Delta$(Length)$_y$ + $\Delta$(Length)$_z$)
Where, $\Delta$(Length)$_x$ = 0 due to no change in length in x direction (axial direction), 
$\Delta$(Length)$_y$ = -$ν$ $\varepsilon \times$ ($L$) , and $\Delta$(Length)$_z$ = -$ν$ $\varepsilon \times$ ($L$)
- $\varepsilon$ is total deformation in steel and $L$ is original length of steel
$\Delta$(Length) = ($\Delta$(Length)$_y$ + $\Delta$(Length)$_z$) = -2$ν$ $\varepsilon$ L
$L_f$= $L_o$ + $\Delta$(Length)
$L_f$ = $L_o$ + $\Delta$(Length)$_y$ + $\Delta$(Length)$_z$
$L_f$= $L_o$ - 2$ν$ $\varepsilon$ L
$L_f$= 1m - 2(0.3)(12.5 x $10^-6$)(1m)
$L_f$ = 0.99947 m

Final area of steel bar is given by,
$A_f$ = ($F$ / Stress)$_f$
$A_f$ = (20 kN / 2.5 kN/$cm^2$)$_f$
$A_f$ = 8000 $cm^2$

Hence, the correct answer is (B) 9.7 $cm^2$.
